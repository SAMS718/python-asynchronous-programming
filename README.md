# 🚀 Mastering Python Concurrency & Asynchronous Programming

A complete beginner-friendly repository to learn how Python executes multiple tasks efficiently using **Threads**, **Processes**, **Coroutines**, and **AsyncIO**.

This repository gradually builds the concepts from basic concurrency to real-world asynchronous programming with HTTP requests, file handling, and email automation.

---

## 📖 Topics Covered

### Part 1 — Python Concurrency Fundamentals

- Understanding Synchronous Programming
- CPU Bound vs I/O Bound Tasks
- Concurrency vs Parallelism
- Introduction to Multi-threading
- Multi-processing
- CPU Core Utilization

---

### Part 2 — Coroutines

- What are Coroutines?
- Coroutine Functions
- Running Coroutines
- AsyncIO Introduction
- Event Loop
- Creating Custom Event Loop
- await Keyword
- Cooperative Multitasking

---

### Part 3 — Asynchronous Programming

- Sequential Coroutines
- Concurrent Coroutines
- asyncio.create_task()
- asyncio.gather()
- Task Scheduling
- Non-blocking Execution

---

### Part 4 — Real World AsyncIO

- Async File Handling
- Async HTTP Requests (aiohttp)
- Async SMTP Email Sending
- API Communication
- Performance Comparison
